# Roadmap / Идеи — Void Drifter

Бэклог фич и план развития. Темп — «средние блоки»: связанная группа фич на
версию `0.0.x.0` (см. схему версий в `CHANGELOG.md`). Номера версий и порядок
ориентировочные — двигаем по желанию.

Легенда статусов: `[ ]` запланировано · `[~]` в работе · `[x]` готово.

---

## План по версиям

### 0.0.1.0 — «Живая арена» (juice, низкий риск)
Быстрые фичи, заметно добавляют веселья и легко тестируются.
- [ ] **Подбираемые предметы** — дропы с врагов:
  - аптечка (восстановить HP),
  - временные бонусы: рапид-фаер, овердрайв (×урон), кратковременная неуязвимость,
    бомба-очистка экрана, магнит притяжения дропов.
- [ ] **Опасности арены**: разрушаемые астероиды как укрытия + телеграфируемые
  лазерные стены, которые надо обходить.
- [ ] **Раздельные рекорды по сложности** — давно напрашивается (сейчас рекорд один
  на все три). QoL, маленький.

### 0.0.2.0 — «Зоопарк врагов»
- [ ] 3–4 новых типа врага. Кандидаты:
  - **Щитовик** — фронтальный щит, неуязвим в лоб, надо зайти сбоку/сзади.
  - **Телепортёр** — мигает к игроку короткими прыжками.
  - **Миномётчик** — навесные снаряды с телеграфом зоны попадания.
  - **Хилер** — лечит соседних врагов (приоритетная цель).
  - **Сплиттер** — при смерти делится на мелких.
- [ ] Расширить таблицу разблокировок по волнам (`CFG.wave.unlock`).

### 0.0.3.0 — «Арсенал»
- [ ] Альтернативные основные пушки (выбор/дроп): дробовик (конус), лазер-луч
  (непрерывный), рельса (пробитие насквозь), бумеранг.
- [ ] Новые модификаторы пуль: заморозка/слоу при попадании, яд (урон по времени),
  рикошет от краёв.
- [ ] Легендарные/эпик-апгрейды с яркими эффектами (редкий тир выше epic).

### 0.0.4.0 — «Боссы»
- [ ] 2–3 уникальных босса со своими паттернами (сейчас один босс-рескин). Идеи:
  - **Близнецы** — две половины, добивать почти одновременно.
  - **Лазерный страж** — вращающиеся лучи + безопасные сектора.
  - **Улей** — призывает рои, уязвим в окнах между волнами призыва.
- [ ] Ротация боссов по номеру волны (вместо одного на всё).

### 0.0.5.0 — «Ангар» (выбор корабля)
- [ ] 3–4 стартовых корабля со своими статами и стилем:
  сбалансированный, быстрый «glass-cannon», танк, дроновод.
- [ ] Экран выбора корабля в меню (рядом с выбором сложности).

### 0.0.6.0 — «Прогресс» (мета между забегами)
- [ ] Валюта за забеги (гемы/кредиты), начисление по очкам/волнам.
- [ ] Постоянные улучшения и разблокировки в меню (новые корабли, стартовые бонусы).
- [ ] Сохранение прогресса: `localStorage` + (на платформе) Yandex player data.

### 0.0.7.0 — «Режимы и рекорды»
- [ ] Режимы: бесконечный, ежедневный челлендж (фиксированный сид), тайматак,
  хардкор (без рероллов/ревайвов).
- [ ] Достижения за подвиги.
- [ ] Таблица лидеров через Yandex SDK (отдельно по сложностям/режимам).

## Детальный бэклог по категориям

Сюда складываем сырые идеи; при планировании версии вытаскиваем нужное наверх.

### Враги и боссы
- Щитовик, телепортёр, миномётчик, хилер, сплиттер, минёр (ставит мины),
  камикадзе-бомбер (заряжается → взрыв).
- Элитные версии обычных врагов (модификатор: быстрее/толще/с аурой).
- Уникальные боссы с фазами и «честными» телеграфами атак.

### Апгрейды и оружие
- Альт-пушки: дробовик, лазер, рельса, бумеранг, ракетница.
- Модификаторы: заморозка, яд (DoT), рикошет, поджог, отбрасывание.
- Эволюции: комбинация двух апгрейдов даёт усиленный эффект.
- Тир «легендарные» с правилами выбора.

### Среда и опасности арены
- Разрушаемые астероиды (укрытия), лазерные стены (телеграф),
  чёрные дыры (притяжение), зоны урона, движущиеся препятствия.
- Вариации арены по «биомам» каждые N волн.

### Подбираемые предметы
- Аптечки, временные бонусы (рапид-фаер, овердрайв, неуязвимость, бомба, магнит),
  валюта (гемы). Балансировать частоту дропа.

### Мета и реиграбельность
- Выбор корабля/персонажа со своими статами.
- Мета-прогресс: валюта → дерево постоянных улучшений/разблокировок.
- Режимы: бесконечный, ежедневный челлендж, тайматак, хардкор.
- Достижения + лидерборд (Yandex). Раздельные рекорды по сложности.

### QoL / полировка (вне блоков, по мере надобности)
- Меню настроек (громкость, флаги эффектов, прицел мышью).
- Раздельные рекорды по сложности (если не вошло в 0.0.1.0).
- Локализация EN (сейчас только RU) — расширяет аудиторию.
